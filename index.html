Engineering rigor

Not approximations.
Real methodology.

Every calculation is based on published industry standards and peer-reviewed engineering references — not simplified rules of thumb.

🔄
Colebrook-White iterative solver
Friction factor solved by 50-iteration Newton-Raphson loop converging to 10⁻¹⁰ — the same algorithm used in commercial simulation software.
📐
API 520 orifice sizing equations
Vapor (critical and subcritical), liquid (Bernoulli with Kw correction), and two-phase (Omega method, API 520 Appendix D) all implemented.
🌡
LMTD-Ft with Bell-Delaware Ft correlation
Ft factor calculated from R and S parameters for 1-2 and 2-4 exchanger configurations, with singularity handling at R=1.
💹
Newton-Raphson IRR solver
Internal rate of return solved by 200-iteration Newton-Raphson on the free cash flow stream, converging to 10⁻⁸ — handles non-standard cash flow profiles.
🏭
Hand factors from Turton et al.
Eight equipment-specific installation cost components (erection, piping, instrumentation, electrical, civil, structural, insulation, painting) per equipment type.
📊
TEMA bundle diameter correlations
Shell diameter estimated from K₁/n₁ layout constants for triangular and square pitch, per TEMA standards for tube bundle geometry.

// Colebrook-White — 50 iterations
function colebrook(Re, eps_D) {
  if (Re < 2300) return 64/Re;
  let f = 0.02;
  for (let i=0; i<50; i++) {
    const fn = 1/(-2*log10(
      eps_D/3.7+2.51/(Re*√f)
    ))**2;
    if(|fn-f| < 1e-10) break;
    f = fn;
  }
  return f; // Darcy friction factor
}
